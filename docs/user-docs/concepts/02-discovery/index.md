## Overview

PML uses multiple search strategies to discover relevant tools:

- **Semantic Search** - BGE embeddings understand natural language intent
- **Hybrid Search** - Combines vector similarity with graph relationships
- **Proactive Suggestions** - Recommends tools based on context and history
