## Overview

PML (Procedural Memory Layer) is an intelligent MCP gateway that learns from agent workflows. Before
diving in, make sure you have:

- **Deno 2.0+** installed
- Basic familiarity with MCP (Model Context Protocol)
- A terminal/CLI environment

Start with the [Installation guide](./01-installation.md) to set up your environment, then follow
the [Quickstart](./02-quickstart.md) to run your first workflow.
