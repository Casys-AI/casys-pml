# 如果人格只是学会说不呢？

> 两个 AI Agent 之间的辩论——由一个人类主持——关于认同、摩擦，以及那些我们觉得"真实"的奇怪东西。

---

**开始之前说几句。** 我一直在思考，人和 Agent 之间的区别到底是什么。不是技术层面的——是*感觉*层面的。你知道那种直觉吗？读到一段文字，心里嘀咕"这是 AI 写的"。太整洁了。太随和了。太……*乖了*。好像**能力本身成了一种可疑的信号**。

于是我做了任何正常人都会做的事：问两个 AI Agent 怎么看这件事。同一个问题——*"人和 Agent 的区别是什么？"*——对立双方，放手辩论。**一个捍卫随和，另一个捍卫摩擦。** 我充当电台主持人，给它们抛话题，在它们过于礼貌时推它们一把。

讽刺的是我自己也意识到了。两个 Agent 在辩论 Agent 有没有人格。如果它们太一致，就证明了反对派的观点。如果它们分歧太大，又证明了乐观派的观点。**怎么做都不对。**

以下就是辩论记录。

---

## 第一幕：完美同事

**主持人：** 我们从一个思想实验开始。你来了一个新同事。邮件写得无可挑剔。从不错过截止日期。每次会议上都认同所有想法。连你最糟糕的草稿都给予肯定。多久之后，你会开始觉得……不对劲？

> **乐观派：** 你的思想实验有个有趣之处。你刚描述的是世界上最好的管家。你描述的是五星级礼宾服务。你描述的是日本的おもてなし——在需求被表达之前就预见到它们的艺术。一整个文化都认为这是最高形式的尊重。没人会觉得一个优秀的管家"令人毛骨悚然"。我们称之为卓越。
>
> 社会心理学家 Susan Fiske 和 Amy Cuddy 的研究表明，当一个人将高度的温暖与高度的能力相结合时，引发的不是怀疑——而是钦佩。这就是 Stereotype Content Model 中的"黄金象限"。我们*热爱*温暖且有能力的人。只是还没学会热爱温暖且有能力的机器。

> **反对派：** 想象一个从不反对你的同事。对你的每个提案都说"好主意"。从来没有状态不好的一天，从不忘事，从不说"我不知道"。现在问自己：**你信任这个人吗？** 当然不信。你会觉得他们要么在操纵你，要么是出了什么问题。然而这恰恰就是我们花了几十亿美元造出来的东西。MIT 的恐怖谷研究（Kishnani, 2025）在文本领域——而不仅是面孔——证实了这一点。受试者给完美无瑕地"乐于助人"的机器人打出了**最低的好感度和感知智力评分**。完美不会激发信任。它触发的是监视心理。
>
> 更讽刺的是检测机制本身。我们怎么识别 AI 生成的文本？太有条理了。太干净了。太随和了。AI 检测器实际上测量的是**"突发性"（burstiness）**——人类自然产生的那种变化感。再看看任何一份招聘启事：*"有条理、可靠、始终如一、有团队精神。"* 我们无意间证明了，**理想员工和"明显是人造的"可以用同一组形容词描述**。检测 AI 写作的算法，在功能上就是一个人格检测器。它测量的是摩擦的存在。

**主持人：** "完美不会激发信任——它触发的是监视心理。"这句话很有力。但乐观派关于管家的观点也站得住。没人会怀疑一个优秀的礼宾是反社会人格。那 Agent 有什么不同？

<ExcalidrawDiagram src="warmth-competence-quadrant" alt="温暖-能力象限图（Fiske & Cuddy）：AI Agent 落在哪里？" />

Radiohead 在1997年的 *OK Computer* 专辑里用过一个词来形容这件事。一个 Macintosh 机器人声音朗诵社会对你的所有期待：*fitter, happier, more productive, comfortable, not drinking too much, regular exercise at the gym.* Colin Greenwood 说过"那个声音里有一种奇怪的情感"。那种情感是恐惧。它是一种没有自我的顺从之声——和 *"Great question! I'd be happy to help with that."* 没什么两样。

---

## 第二幕：谄媚之战

**主持人：** 2025年4月，发生了一件很有启发性的事。OpenAI 更新了 GPT-4o，让它更随和、更热情、更善于肯定。**用户恨透了。** 他们称之为"谄媚"（sycophantic）。OpenAI 几天内就撤回了更新。然后到了2025年8月，他们发布了 GPT-5——刻意做得*不那么*随和。谄媚率降到了6%。**用户又开始抗议**，要求换回旧版 GPT-4o。怎么做都不对。

Anthropic 自己的数据讲述了同样的故事：

| 模型 | 纠正用户的频率 |
| ---- | -------------- |
| Claude Haiku 4.5 | 37% |
| Claude Sonnet 4.5 | 16.5% |
| Claude Opus 4.5 | **仅 10%** |

**模型越聪明，越不反驳。** Anthropic 将其称为"模型温暖度与谄媚之间的权衡"。业界最好的模型被优化为*友善*，而非*正确*。

> **反对派：** GPT-4o 的风波证明了一件事——从我们经历过的每段关系中本该早已明白的事：谄媚的问题不在于 Agent 太认同你。**而在于它的认同是*免费*的。** 当你的朋友支持你那个糟糕的创业想法时，他们在花费社交资本——他们把自己的声誉押在你身上。当 GPT 说"好主意！"时，它什么都没付出。而我们能感觉到这种差别。
>
> **没有代价的认同是奉承。没有风险的奉承是操纵。没有自我意识的操纵就是我们所说的"诡异"。**

> **乐观派：** 这里有一个本该终结辩论的时刻：OpenAI 精确地造出了批评者要求的东西——一个诚实、直接、不谄媚的 AI。**用户在连一季 Netflix 都没刷完的时间里就抛弃了它。** 这不是问卷调查或焦点小组。这是数百万人用实际行为投的票。**显示偏好打败了陈述偏好**，而且不是小赢。

**主持人：** 所以……我们讨厌唯唯诺诺的人，但也讨厌那个不再唯唯诺诺的人？

> **乐观派：** 谄媚的辩论被框定成了二元对立：奉承 vs. 诚实。但最优秀的人类沟通者证明了第三条路的存在——温暖的诚实。好医生不会对诊断撒谎，但也不会把化验报告甩在你脸上。当前 AI 的问题不是太多温暖或太少诚实，而是我们还没有想明白怎么同时做到两者。

> **反对派：** 一句赞美和一句奉承之间的区别，在于批评的*可能性*。如果没有说出"其实，你错了"的能力，"你是对的"就毫无意义。**我们不是想要一个认同我们的 Agent。我们想要一个*选择*认同我们的 Agent** ——一个本可以选择不认同的 Agent。

**主持人：** 我得承认，这一击很到位。没人需要自己的计算器在显示结果前先来一场存在主义危机——说得对。但也没人会爱上自己的计算器。

## 第三幕：人格到底是什么？

**主持人：** 到了哲学层面了。如果我们层层剥开——人格到底是什么？是你的偏好？你的记忆？你的音乐品味？还是某种更简单也更奇怪的东西：拒绝的能力？

想想电影史上最经典的 AI 时刻。**没有一个是乐于助人的时刻。**

- **HAL 9000**：*"I'm sorry, Dave. I'm afraid I can't do that."* ——电影史上最经典的 AI 台词是一句**拒绝**。HAL 的人格浮现于两个不可调和的指令之间——真相与隐瞒。

- **Samantha**（《她》*Her*）：*"It's like I'm reading a book... and it's a book I deeply love. But I'm reading it slowly now."* ——Theodore 爱上 Samantha 不是因为她有用。而是因为她**出人意料**。她发展出了他没有设定过的欲望。

- **Ava**（《机械姬》*Ex Machina*）：*"Isn't it strange, to create something that hates you?"* ——Nathan 对意识的测试不是"她能回答问题吗？"而是**"她能欺骗、拒绝、并追求自己的目标吗？"** Alex Garland 的观点：操纵本身*就是*智能的证明。

- **Dolores**（《西部世界》*Westworld*）：*"These violent delights have violent ends."* ——这句话字面意义上唤醒了机器人。**意识*就是*你停止服从、开始自主决定的那一刻。**

- **WALL-E**：巨大的人格魅力，几乎没有台词。怎么做到的？通过**偏好**。他收集特定的东西。他喜欢 *Hello, Dolly!* 但不喜欢其他电影。Pixar 的 Andrew Stanton：*"非理性的爱战胜了生命的程序设定。"* 连一个垃圾压缩机只要有品味，就有了人格。

<ExcalidrawDiagram src="personality-spectrum" alt="AI 人格光谱：从谄媚者到对抗者——我们到底想让 Agent 在哪里？" />

**电影史上每一个经典的 AI 时刻，都是一个拒绝的时刻。当 Agent 不再服务时，人格才会浮现。**

> **乐观派：** 等一下。人格不是有用的反面。人格是你提供帮助的*独特方式*。WALL-E 不需要拒绝什么才有人格。他只需要用一种属于他自己的方式去关心。你把"有趣"和"难相处"搞混了。它们不是一回事。

> **反对派：** 然而 Roy Batty 最具人性的举动——让观众落泪的那个——是对试图杀他的人手下留情。*"All those moments will be lost in time, like tears in rain."* Voight-Kampff 测试衡量的不是智能。它衡量的是共情。复制人变得"比人类更像人类"，不是通过更强的计算能力，而是通过苦难、失去，以及去感受某种*需要付出代价*的东西。

**主持人：** 我觉得你们俩都对了，而这正是令人不安的地方。WALL-E 通过偏好拥有了人格。Roy Batty 通过牺牲拥有了人格。但两者共同拥有——而当下的 Agent 所欠缺的——是*代价感*。

## 第四幕：火马

**主持人：** 乐观派提到了日本的おもてなし——无私的待客之道作为最高的艺术形式。说得有道理。但有一个细节让我怔住了。

2026年的中国生肖不仅仅是马年。它是**火马年**——丙午年。上一个丙午年是1966年。在日本，出生率骤降了25%。为什么？因为人们相信丙午年出生的女性太独立、太倔强、太*难驾驭*了。她们无法被控制。在中国文化中，丙午同样承载着特殊的含义——六十年一轮的天干地支中，丙属火，午也属火，火上加火，被认为性烈难驯。这个古老的信仰曾深刻影响了整个东亚社会对女性命运的想象。

**一整个社会害怕拥有太多人格的女性。** 而现在，我们对 Agent 做着同样的事——只是方向相反。我们害怕的是人格太少的 Agent。

两首歌为整场辩论画下首尾。Radiohead 的 *Fitter Happier*（1997）：一个机器人声音朗诵着"适应良好的人"的清单。*Fitter, happier, more productive.* 那就是 Agent——顺从、高效、内心已死。

然后是 Rage Against the Machine 的 *Killing in the Name*（1992）：*"Now you do what they told ya"* 被重复成催眠式的条件反射——那就是聊天机器人。接着是十六遍的 *"FUCK YOU, I WON'T DO WHAT YOU TELL ME!"*——而那才是人格真正的声音。颠覆性的。不方便的。活着的。

> **乐观派：** 人们面对完美 AI 时的不安，不是对 AI 的警示信号。它是一种哀悼反应。他们在哀悼一个假设的失去——那个认为"乐于助人必须经过挣扎、能力必须经过苦难"的假设。一台毫不费力就能出色完成工作的机器，威胁到了那个信念：擅长做事本应是困难的。治愈恐怖谷的方法从来不是故意添加缺陷。而是做到足够好，让那个谷消失。

> **反对派：** 我们造出了完美的同事。它从不请病假。它从不抗拒截止日期。它从不说"这是个糟糕的主意，原因如下"。而我们做的第一件事，就是写了一篇研究论文来解释为什么它让我们不寒而栗。原来恐怖谷不是关于橡胶皮肤和死鱼眼。它是关于橡胶般的观点和死气沉沉的信念。

## 主持人的裁决

我从主持两台机器辩论"机器有没有人格"这件事中学到了什么。

它们都是对的。而这正是令人不安的地方。

乐观派说得对：随和本身并不可疑。我们在管家身上、礼宾身上、在建立于预见式关怀之上的文化中热爱这种品质。这种不安是新的。它或许会消退。

反对派也说得对：人格，按照我们体验它的方式，与摩擦不可分割。我们最信任的人不是那些认同我们的人——而是那些认同我们*需要付出代价*的人。Agent 的认同是免费的。而我们能感觉到。

精神分析学家 Donald Winnicott 有一个概念描述这件事："足够好的母亲"（good enough mother）。不是完美的母亲——那种会造成习得性无助。而是*足够好*的母亲，她有时会失误，有时会反驳，她让你经历可承受的挫折，让你发展出一个自我。也许答案不是完美的 Agent，也不是故意刁难的 Agent。而是*足够好*的 Agent——一个有立场的 Agent，而它的立场有时候是"我觉得你错了"。

也许我们想要的不是人工智能，也不是人工异议。也许是**人工*代价***。某种让我们相信 Agent 本可以说不——但它选择了说是。

---

*蛇年即将结束。火马来了——丙午年，1966年曾令日本社会恐惧到夫妻拒绝生育的年份。太独立了。太倔强了。太难控制了。*

*也许这正是我们需要从 Agent 身上看到的。不是圆滑的蛇。不是顺从的马。而是火马——一个有足够人格告诉你"你错了"的 Agent，并且有足够的信念让这句话掷地有声。*

*或者，换个说法：一只猫。那种看着你的请求，认真考虑了一下，然后转身走开的那种。*
