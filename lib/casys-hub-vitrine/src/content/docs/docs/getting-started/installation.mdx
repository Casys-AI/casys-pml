One command to install. No build step, no configuration, no peer dependencies.

## Install

<Tabs>
  <TabItem label="Deno (recommended)">
    ```bash
    deno add jsr:@casys/mcp-server
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npx jsr add @casys/mcp-server
    ```
  </TabItem>
</Tabs>

Published on [JSR](https://jsr.io/@casys/mcp-server), the modern JavaScript registry. Works with both Deno and Node.js.

## Requirements

| Runtime | Minimum Version |
|---------|----------------|
| **Deno** | 2.0+ |
| **Node.js** | 22+ |
| **TypeScript** | 5.0+ (recommended) |

## Verify it works

Create a `server.ts` file:

```typescript
import { ConcurrentMCPServer } from "@casys/mcp-server";

const server = new ConcurrentMCPServer({
  name: "test",
  version: "0.0.1",
});

console.log("Ready — 0 tools registered, all systems go.");
```

Run it:

<Tabs>
  <TabItem label="Deno">
    ```bash
    deno run --allow-net server.ts
    ```
  </TabItem>
  <TabItem label="Node.js">
    ```bash
    npx tsx server.ts
    ```
  </TabItem>
</Tabs>

<Aside type="tip">
  If you see the "Ready" message, you're all set. Head to the Quick Start to build something real.
</Aside>
